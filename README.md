## The Last of Guss

### Установка

1. Установите зависимости:
```bash
npm install
```

2. Скопируйте файл окружения:
```bash
cp .env.example .env
```

3. Заполните переменные в файле `.env`

4. Сборка
```bash
npm run build
```

5. Запустите
```bash
npm run start
```

6. Откройте в браузере.

http://localhost:3000/
